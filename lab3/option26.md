## Вариант 26: Расчет индекса массы тела (ИМТ)

### Описание функции
Расчет ИМТ и определение категории веса.

### Входные параметры
| Параметр | Тип | Диапазон/Допустимые значения |
|----------|-----|------------------------------|
| `weight` | float | 20 - 300 кг |
| `height` | float | 100 - 250 см |

### Ожидаемое поведение (бизнес-правила)
- ИМТ = weight / (height/100)^2
- < 18.5: "Недостаточный вес"
- 18.5 - 25: "Нормальный вес"
- 25 - 30: "Избыточный вес"
- 30 - 35: "Ожирение I степени"
- > 35: "Ожирение II степени"

### Формат результата
```python
{
    "bmi": float,  # индекс массы тела
    "category": str  # категория
}
```

### Исходный код
```python
def calculate_bmi(weight, height):
    height_m = height / 100
    bmi = weight / (height_m ** 2)
    
    if bmi < 18.5:
        category = "Недостаточный вес"
    elif bmi <= 25: 
        category = "Нормальный вес"
    elif bmi <= 30:
        category = "Избыточный вес"
    elif bmi <= 35:
        category = "Ожирение I степени"
    elif bmi > 35:
        category = "Ожирение II степени"
    
    return {"bmi": round(bmi, 1), "category": category}
```
