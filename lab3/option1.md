## Вариант 1: Калькулятор страховки

### Описание функции
Модуль расчета стоимости страхового полиса для автомобиля с учетом возраста водителя, стажа и мощности двигателя.

### Входные параметры
| Параметр | Тип | Диапазон | Ограничения                          |
|----------|-----|----------|--------------------------------------|
| `driver_age` | int | 18-80 | -                                    |
| `experience` | int | 0-60 | Не может превышать `driver_age - 18` |
| `engine_power` | int | 50-500 | -                                    |

### Ожидаемое поведение (бизнес-правила)
- Базовая ставка: 2000 руб.
- Если возраст < 25 лет И стаж < 3 лет: коэффициент 2.5
- Если возраст >= 25 лет И стаж < 2 лет: коэффициент 2.0
- Если возраст >= 35 лет И стаж >= 10 лет: коэффициент 0.8
- Если мощность > 250 л.с.: дополнительный коэффициент 1.6 (применяется после возрастных коэффициентов)
- Итоговая стоимость = базовая ставка × возрастной коэффициент × коэффициент мощности

### Исходный код
```python
def calculate_insurance(driver_age, experience, engine_power):
    base_rate = 2000
    age_coef = 1.0
    power_coef = 1.0
    
    if driver_age < 25 and experience < 3:
        age_coef = 2.5
    elif driver_age >= 25 and experience < 2:
        age_coef = 2.0
    elif driver_age > 35 and experience >= 10:
        age_coef = 0.8
    
    if engine_power >= 250:
        power_coef = 1.6
    
    return base_rate * age_coef * power_coef
```
